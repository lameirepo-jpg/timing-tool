**Features:**

**Speaking Section:**
- Listen and Repeat: 8-minute timer (for 7 sentences)
- Take an Interview: 3-minute timer (4 questions × 45s each) with +45s skip button

**Writing Section:**
- Build a Sentence: 5-minute timer (10 sentences)
- Write an Email: 7-minute timer
- Write for Academic Discussion: 10-minute timer

**Quick Start Options:**
- Speaking Section (8 min)
- Writing Section (~17 min)
- Full Practice (~25 min)

**Audio Alerts:**
- Start beep when timer begins
- Warning beeps at 30 seconds and 10 seconds
- Countdown beeps for last 3 seconds
- Time-up melody when timer ends
- Audio toggle button (top-right) to enable/disable

**Visual Features:**
- Large countdown display with progress bar
- Color changes: yellow at 30s, red at 10s
- Status indicators (active/paused/stopped)
- Pulsing animation when time is low

**How to Use:**
1. Open `toefl_timer.html` in any web browser
2. Click "Start" on any task to begin
3. Use "Pause" to pause, "Reset" to restart
4. Use quick start buttons for full section practice
5. Try to finish 5-10 seconds before time is up!
